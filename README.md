### Published Work

**[Claude's Hidden Toolkit](https://github.com/N1-AI/claude-hidden-toolkit)** — 100+ page reverse-engineering of Claude.ai's 28 internal tools. Detailed technical cards, cross-platform verification (browser / desktop / mobile), behavioral testing. Published under N1AI. The first systematic documentation of its kind.

**[Hardstop](https://github.com/frmoretto/hardstop)** — Fail-closed safety plugin for Claude Code and Cowork. Two-layer defense: 170 regex patterns + Haiku semantic analysis. `npx hardstop install`

**[Clarity Gate](https://github.com/frmoretto/clarity-gate)** — Pre-ingestion verification for RAG epistemic quality. +19-25% improvement benchmarked on Gemini 3 Flash and GPT-5 Mini (5 models, 39 traps).

**[Stream Coding](https://github.com/frmoretto/stream-coding)** — Documentation-first AI development methodology. The parent framework. Thesis: "LLMs don't need help — they need documentation."

**[specverify](https://github.com/frmoretto/specverify)** — Independent specification verification for AI-ready documents. Two-pass protocol (structural + analytical) catching issues self-certification misses. Framework-agnostic. Runs as a separate-session gate before code generation.

**[coderec](https://github.com/frmoretto/coderec)** — Brownfield companion to Stream Coding. Produces a Codebase Intelligence Document through four analytical layers (cartography, module intelligence, cross-module synthesis, decision archaeology). Framework-agnostic. Makes existing codebases legible to AI.

**[ArXiParse](https://arxiparse.org)** — Three-tier scientific paper extraction platform. Multi-provider LLM support.

**[Claude Memory Research](https://frmoretto.github.io/claude-memory-user-edits-guide/)** — Systematic testing of Claude's undocumented memory system. 68K views, 215 shares on Reddit.
